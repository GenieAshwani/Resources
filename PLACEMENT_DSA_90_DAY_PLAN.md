# Placement DSA Sheet for Students

This document is made for students who want to crack placement interviews step by step.

We divide preparation into 3 levels:

1. `Level 1`: Service-based companies
2. `Level 2`: Product-based companies
3. `Level 3`: Tier-1 companies

Total questions in this sheet: `300`

Distribution:

- `100 questions` for Level 1
- `100 questions` for Level 2
- `100 questions` for Level 3

Recommended rule:

- Solve questions in order
- Write solution in your own language first
- Then code it
- After coding, dry run with one example
- Revise wrong questions every weekend

---

## How Students Should Practice

- First solve without seeing solution for `20-30 minutes`
- If stuck, look at hint only
- Then write full code
- Practice on platforms like `LeetCode`, `GeeksforGeeks`, `Coding Ninjas`, and `HackerRank`
- Maintain one notebook for patterns and mistakes
- Revise all wrong questions at least `3 times`

---

## Final Advice

If a student is very weak in DSA:

- Complete Level 1 first
- Then do only selected important questions from Level 2
- Then move to Level 3 only after confidence improves

If a student is already comfortable:

- Do daily `1 easy + 2 medium + 1 hard`
- Give one mock every weekend
- Focus on explaining approach clearly in interview language

This sheet is enough to build a strong placement preparation roadmap for `90 days`.
